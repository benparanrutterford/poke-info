# poke-info
A tiny program that displays information about Pokémon.

## Usage

Once the CLI has been built, it can be executed via the `poke-info` file in the `bin/` directory:

```
./bin/poke-info --help
```

The `PATH` environment variable can be updated to include the `poke-info` binary:

```
export PATH=$PATH:$(pwd)/bin
```

allowing the CLI to simply be used like so:

```
poke-info --help
```
